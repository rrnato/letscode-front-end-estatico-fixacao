Tag "< img >":
A tag img é utilizada para inserção de uma imagem, não necessita de fechamento, precisa vir acompanhada do "scr" ;
Tag "< p >"":
A tag p é utilizada para adicionar um paragrafo;
Tag "< ol >":
A tag ol é utilizada para inserção de uma lista de itens ordenados;
Tag "< ul >":
A tag ul é utilizada para inserção de uma lista de itens desordenados;
Tag " < li >":
A tag li é utilizada dentro das listas formadas pelas tags ol e ul, com elas que incluimos os elementos da lista;
Tag "< input >":
A tag input é utilizada para preenchimento e interação com os usuários;
Tag " < b >":
A tag b é utilizada para deixar o texto negrito;
Tag " < strong >":
A tag strong também é utilizada para deixar o texto negrito, porém, em questão de acessibilidade, ela faz o texto ser lido com entonação mais forte.;
Tag "< i >":
A tag i é utilizada para deixar o texto em itálico;
Tag " < br >":
A tag br é utilizada para dar uma quebra de linha, não é necessário utilizar tag de fechamento;
Tag " < a >":
A tag a é utilizada como âncora, com ela conseguimos ligar a algum outro lugar da página html.
